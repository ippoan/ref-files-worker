# src/types/

**Do not hand-edit files in this directory.**

Everything here is generated by `ref-files-mcp-server-rs`'s `gen-ts` binary
(`cargo run --bin gen-ts`) and copied here by
`.github/workflows/sync-types.yml`. The Rust source of truth lives at
[`ippoan/ref-files-mcp-server-rs/src/types/`](https://github.com/ippoan/ref-files-mcp-server-rs/tree/main/src/types).

To regenerate locally:

```bash
git clone https://github.com/ippoan/ref-files-mcp-server-rs ../ref-files-mcp-server-rs
(cd ../ref-files-mcp-server-rs && cargo run --bin gen-ts)
cp ../ref-files-mcp-server-rs/ts-bindings/*.ts ./src/types/
```

CI will fail the PR if the copy here doesn't match what the Rust crate would
emit at the SHA pinned in `sync-types.yml`.
